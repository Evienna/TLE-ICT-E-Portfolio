Q3 Lesson 1: Formatting Text with CSS: Typography

          

Broswer Output:

    


What Was Learned


What I Learned:
          
I learned that Typography in CSS involves using properties to control the arrangement and styling of text for better legibility and visual appeal.
Applying the lessons from the PDF, I practiced:
          
Font Priority: I learned to use font-family to create a prioritized list of fonts, ensuring the browser has a fallback if the first font 
is missing.
          
Font Weight: I discovered that font-weight values range from 100 (Thin) to 900 (Ultra Bold), and font-size can be set in pixels (px),
points (pt), or percentages (%).
          
Text Manipulation: I learned how to use font-variant: small-caps to make lowercase letters appear as smaller uppercase ones and 
text-transform to force text into uppercase or lowercase.
          
Spacing Control: I learned that letter-spacing controls the horizontal space between individual characters, while word-spacing adjusts 
the space between words.
          
Visual Effects: I learned to apply text-shadow by defining the horizontal offset, vertical offset, blur radius,
and color.
          
White Space Management: I learned about properties like white-space: pre and pre-wrap which allow me to control how the browser
handles line breaks and sequences of spaces.


Q3 Lesson 2: More Formatting Fundamentals 

          

Broswer Output:

    


What Was Learned


What I Learned:
I learned and reviewed the fundamental CSS properties used to format text blocks and individual characters. Specifically:

Indentation: I learned that the text-indent property allows me to indent only the first line of a paragraph, 
with measurements in px, em, or %.

Alignment: I reviewed how to use text-align to set text to the left, right, center, or justify it for a 
professional look.

Case Control: I learned that text-transform can be used to quickly switch text between uppercase, lowercase, and 
capitalize without changing the HTML.

Spacing & Backgrounds: I learned to adjust the vertical space between lines using line-height and how to apply a
background color specifically to text elements to make them stand out.
          


Q3 Lesson 3: Layout, Positioning, and Backgrounds 

          

Broswer Output:

    


What Was Learned


What I Learned:
I learned how to manage the layout and positioning of elements on a webpage using the CSS Box Model and positioning schemes.
My key takeaways include:

The Box Model: I learned that every element is essentially a box consisting of margins, borders, padding, and the actual content.

Positioning Schemes: I learned the four main ways to position objects: static (default), relative (offset from normal position),  
absolute (placed relative to a reference point), and fixed (stays in place during scrolling).

Floating & Wrapping: I learned to use the float property to move elements to the left or right, allowing text to wrap around them.

Background Control: I learned to set background images, control their repetition (repeat, no-repeat), and define whether 
they scroll with the page or stay fixed using background-attachment.

Margins: I learned to set specific spacing around elements using margin-top, margin-right, margin-bottom, and margin-left.
