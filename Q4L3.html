What Was Learned

I learned that JavaScript is an object-oriented, client-side scripting language that allows developers to manipulate the browser 
and its content through built-in objects. The core components include:

Window Object: The main browser object containing properties like location and history.

Document Object: Represents the HTML content; used to change background colors (bgColor) or write text directly to the page.

Math Object: Provides mathematical constants like Math.PI and methods like Math.max() or Math.pow().

Navigator Object: Used to detect browser details such as the application name (appName) and operating system (platform).

Data Types: JS variables can hold various types, including Strings (text), Integers (whole numbers), Floats (decimals), and Booleans (true/false).

Variable Rules: Names must be case-sensitive, cannot start with a digit, and should avoid using JavaScript reserved words.
