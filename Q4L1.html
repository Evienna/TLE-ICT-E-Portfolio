Q4 Lesson 1: Introduction to JavaScript and Web Dynamics


Core Purpose
Interactivity: Creates dynamic user experiences (e.g., Facebook’s UI, real-time updates).

Functionality: Handles content loading, video controls, and user tracking (Meta Pixel).

Static vs. Dynamic Web Pages
Static: "Flat" pages delivered exactly as stored (e.g., a photographer’s portfolio).

Dynamic: Content changes based on user actions or variables (e.g., social media feeds).

What is JavaScript (JS)?
The "Big Name" in Web: The most popular scripting language; works in all major browsers.

History: Created by Brendan Eich (Netscape) as Mocha, then LiveScript, then finally JavaScript.

Tech Specs:

Client-side: Processed on the user's computer, not the server.

High-level: Written in "English-like" words for readability.

Interpreted: Converted directly from source code to machine code at runtime.

Object-Oriented: Based on manipulating "objects."

Rules: Case-sensitive; semicolons are good practice but not strictly required.

Core Capabilities
Event Reaction: Executes code based on triggers like mouse clicks or page loading.

Form Validation: Checks user input locally to save server processing time.

Cookies: Saves/retrieves info on the user's computer to monitor preferences/habits.

Key Features
C-Syntax Support: Uses structured programming similar to the C language.

Dynamic Typing: Variable types can change during execution based on assigned values.

Local Execution: Runs directly in the browser for faster, more responsive interaction.

DHTML: Combine JS + CSS to create flexible effects (scrollers, mouse trails).

Pros and Cons
The "Essential" Terms
Script: A set of instructions connecting components to perform a task.

JS Essentials: Requires only a text editor and a web browser.

3 Pillars of JS: Everything is built on Objects, Methods, and Properties.

Implementation Basics
The Tag: Use the script tag to tell the browser you are using JavaScript.

The Attribute: Use type="text/javascript" to define the version.

The First Command: document.write() is the standard command used to display output on a webpage.
